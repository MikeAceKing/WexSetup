Wexio Desktop Runtime

Download Wexio Setup

Install the native Wexio desktop runtime for Windows, macOS, and Linux. The desktop app includes the local bridge used by Wexio OS to open approved apps, files, links, and system tools from the workspace.

  • Use this domain for installer delivery and desktop runtime onboarding.
  • The main product and tenant workspace stay on the Wexio app domain.
  • The desktop runtime enables the native in-Wexio browser plus the localhost desktop bridge.

Deployment status

This repository is intentionally static. Railway should deploy it through the included Dockerfile, while GitHub Actions builds and republishes the native installers.

Bridge-enabled installers The published installers now package the Wexio desktop runtime together with the localhost bridge used by Wexio OS desktop integrations.

Public Railway URLs should stay on HTTPS only. Do not use :8080 from outside Railway.
